# ChatGLM-6B

网址：[https://github.com/THUDM/ChatGLM-6B](https://github.com/THUDM/ChatGLM-6B)

模型：[https://huggingface.co/THUDM/chatglm-6b](https://huggingface.co/THUDM/chatglm-6b)

## 下载环境

拉代码

```bash
git clone git@github.com:THUDM/ChatGLM-6B.git
```

通过conda 创建虚拟环境

```bash
conda create -n llm python=3.10
```

激活环境

```
conda activate llm
```

进入ChatGLM-6B文件夹

```
pip install -r requirements.txt
```

## 下载模型

手动去 hugging face 下载吧，然后再传上来。
